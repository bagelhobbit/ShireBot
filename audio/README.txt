Creating .dca files from .wav files (Windows/MYSYS2)

DCA specification can be found at https://github.com/bwmarrin/dca
Code/instructions from https://github.com/jonas747/dca/tree/master/cmd/dca
Requires Go and ffmpeg to be installed and in the system path.

Using MSYS2:
run `dca.exe in.wav > out.dca`

Input file can also be piped to dca.exe.
